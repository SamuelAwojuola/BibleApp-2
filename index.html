    Shortcut Keys

    1. Alt + V: Detach Side Windows
    2. Alt + C: Open/Close Compare-Search Window
    3. Alt + T: Toggle Showing Hebrew/Greek Lemma Transliteration
    4. Alt + Z: Detach Versnote and Center it on the page.
    5. Alt + H: Toggle highlighting hovered version
    6. Alt + S: Scripture Reference Input
    7. ALT + Q: Loop through the Fonts and Color Themes
    8. Ctrl + Shift + D: Toggle highlighting of all words with the same Strong's number
    9. Ctrl + Shift + F: Search for highlighted text
    10. Ctrl + Shift + Z: Toggle Side Nav does not work if any input has the focus.
    11. Ctrl + Up Arrow Key: Go to previous chapter
    12. Ctrl + Down Arrow Key: Go to next chapter
    13. n/N: Show/Hide VerseNote for most recently clicked verse
    14. m/M: Show/Hide Markers for most recently clicked verse
    15. x/X:
      1. With Mouse Over RightClick Menu: Show/Hide CrossRef and Versions Buttons in Scripture RightClick Window
      2. When Mouse Is Not Over RightClick Menu: Show/Hide CrossRefs for most recently clicked verse
    16. Escape: Toggle Showing side and top bar and their contents

    Search

      Search Modifiers

    1. Search for any of the words: [word1 word2 word3 ...]
    2. * Indicate Part of Word That Could Be Anything
    3. Ends with word: *word1
    4. Starts with word: word2*
    5. Whole word: *word3*
    6. Phrase (Whole and Partial Words of Phrase): "word1 word2* *word3 ..." Surround with double quotes `"` to search for phrase
    7. The above works within brackets: [*word1 word2* *word3* "word4 word5* *word6..." ...]

      Short Cut Keys

    1. Search for highlighted text: {Ctrl + Shift + F}
    2. Open Search Section/Window: {Right Click} on Search Button or SearchSettings Button in Top Bar
    3. Clear and Close Search Section/Window: {Right Click} or {CTRL + Click} the Close (x) Button
    4. Where to Run Search: Search Window or Compare Verses Window (Alt + C)
    5. Strong's Analysis: Return only strong analysis without verses text{Shift + Search (Enter or Click)}

    ContextMenu Search Icon

    1. Under "Word Relatives" Section
      1. Search Icon/Button at the begining of a list gets all the Strong's Number in that list item, including those in its children list.
      2. Search Icon/Button before a Strong's Number only gets that Strong's Number
    2. Left Click: Search for Strong's Number
    3. CTRL + Left Click: Add Strong's Number to Search Input Without Clearing It and Run Word Search
    4. Right Click: Add Strong's Number to Search Input Without Clearing It and Make It Ready for Enter Key Press
    5. CTRL + Right Click: Clear Search Input and Paste Strong's Number and Make It Ready for Enter Key Press

    Books & Chapters Search Ranges

    1. Use full book names or standard abbreviations
      1. Revelation 2: Search in Revelation 2
      2. Rev2: Search in Revelation 2
    2. Separate beginning and end of rang with a dash
      1. Gen-Rom: From Genesis to Romans
      2. Rut - Gen: Search backwards from Ruth to Genesis
      3. 1Sam3-10: Search from 1Samuel 3 to 1Samuel 10
      4. Gal 6-1: Search backwards from Galatians 6 to Galatians 1
    3. Seperate list of books/ranges with semicolon
      1. Gal6,Rom2-6: Search Galatians 6 and Romans Chapter 2 to Chapter 6
      2. Gal6;Rom2-6: Search Galatians 6 and Romans Chapter 2 to Chapter 6

    For Compare Window/Sections

    1. Use full book names or standard abbreviations
    2. Reference must at least have Book name and chapter number
    3. Rev2: Will generate all of Revelation Chapter 2
    4. Seperate list of books/ranges with semicolon
    5. Gal6;Rom2:1-6: Search backwards from Galatians 6 to Galatians 1
    6. Gal6:1,3,10;Rom2:1,2,9,5: Use commas to separate verses
    7. Range list can include words to search for
    8. Zelophehad;Gal6;Rom2-6;Nazareth
    9. Chapter Range, e.g., 1Sam3-10 Not Yet Supported
